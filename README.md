# Ocean Behavioral Group — Website

Ready-to-deploy React website (Vite).

## Deploy free with Vercel (no coding needed)

1. Create a free account at github.com
2. Create a new repository (e.g. "obg-website") and upload ALL files in this folder
   (keep the folder structure: src/ folder inside)
3. Create a free account at vercel.com — sign in with your GitHub
4. Click "Add New Project" → Import your "obg-website" repository
5. Vercel auto-detects Vite. Click Deploy.
6. In ~1 minute your site is live at: obg-website.vercel.app

## Custom domain (optional)

1. Buy a domain (e.g. oceanbehavioralgroup.com) at Namecheap, GoDaddy, or Google Domains (~$12/yr)
2. In Vercel → your project → Settings → Domains → Add your domain
3. Follow the DNS instructions Vercel shows you (usually 2 records to add)

## Run locally (optional, for developers)

npm install
npm run dev
